# PayPal Certification - Screenshot Requirements
## Simple Checklist

---

## What PayPal Wants to See:

### **1. PayPal Logos & Buttons** (Prove logos are from official sources)
- [ ] PayPal button on checkout page
- [ ] Venmo button on checkout page  
- [ ] Card payment fields (PayPal-hosted)
- [ ] PayPal messaging/financing text

---

### **2. Seller Dashboard** (Prove sellers can manage PayPal connection)
- [ ] Connected status (showing merchant ID, email, "can receive payments")
- [ ] Not connected status (showing "Connect PayPal" button)
- [ ] Disconnect confirmation dialog
- [ ] Refresh status button

---

### **3. Pay Later Settings** (Prove sellers can enable/disable Pay Later)
- [ ] Toggle switch for "Enable Pay in 4"
- [ ] Toggle in ON position
- [ ] Toggle in OFF position

---

### **4. Refund System** (Prove sellers can issue refunds)
- [ ] Transaction list page
- [ ] Transaction detail page with "Issue Refund" button
- [ ] Refund modal/form (amount, reason fields)
- [ ] Insufficient funds error message
- [ ] Successful refund confirmation

---

### **5. PayPal Messaging** (Prove Pay Later messaging is displayed)
- [ ] Messaging on service/product pages
- [ ] Messaging on checkout page
- [ ] "We accept PayPal and Venmo" banner (if you have one)
