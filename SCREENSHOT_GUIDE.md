## üéØ Tips for Great Screenshots

1. **Use Full HD Resolution** (1920x1080 or higher)
2. **Full Page Screenshots** - Use browser extension like "Full Page Screen Capture"
3. **Hide Personal Info** - Blur out real emails, phone numbers
4. **Clean Browser** - Hide bookmarks bar, close unnecessary tabs
5. **Test Data** - Use demo/test accounts with fake but realistic data
6. **Good Lighting** - Clear, readable text
7. **Annotations** - Add arrows/highlights if needed (use tool like Snagit)
8. **Consistent Format** - All PNG or all JPG
9. **Name Clearly** - Follow naming convention above
10. **Double Check** - Review each screenshot before moving on
